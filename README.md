# Quick Math

Fast-paced mental arithmetic game built with Blazor WebAssembly.

Practice addition, subtraction, multiplication, and division against the clock. Build streaks, earn points, and beat your high score.

## Play

Visit the [Website](https://pikmintea.github.io/QuickMath/):  [pikmintea.github.io/QuickMath](https://pikmintea.github.io/QuickMath/)

## Run locally

```
dotnet run
```

Opens on `http://localhost:5267`.

## Tech

- Blazor WebAssembly (.NET)
- Bootstrap 5
